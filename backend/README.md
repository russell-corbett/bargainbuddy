# Backend README

Backend API for Project

## Stack

- TypeScript
- Express
- Prisma
- SQLite

## Commands

 | Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:8500/api`  |
| `npm run lint`            | Run eslint                                       |
| `npm run migrate`         | For pushing table changes to database            |
